# CSGO-Configs
Config Files for CS:GO - Includes custom execs for custom gamemodes.<br/>
Includes autoexec file which includes all my configuration such as:

  • Viewmodel          
  • Crosshair          
  • Sound Config       
  • Basic Keybinds     
  • Nade Keybinds      
  • Buy Menu Binds    
  • Shortened Commands
