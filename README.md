# CSGO Configs

**Config Files for CS:GO - Includes custom execs for custom gamemodes.<br/>**

Download: [Available on `60%` Branch](https://github.com/PINPAL/CSGO-Autoexec/tree/60%25)

• Viewmodel<br/>
• Crosshair<br/>
• Grenade Crosshair<br/>
• Sound Config<br/>
• Basic Keybinds<br/>
• Grenade Keybinds<br/>
• Buy Menu Binds<br/>
• Shortened Commands<br/>
